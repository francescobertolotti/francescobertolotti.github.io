# Recurring Exam Style

## Recurring Text Structure
- Opening scenario with data and applied context (physical measurements, consumption, time series).
- Sequence of tasks in a single narrative block, with logical order: calculation -> chart -> interpretation.
- Closing with requests for qualitative comments on numerical results.

## Observed Stylistic Traits
- Explicit references to worksheet cells: about **157** occurrences.
- Requests for charts positioned in a precise area: about **53** occurrences.
- Requests for textual explanation ("explain/comment"): about **57** occurrences.
- Formula-output policy (in the initial instructions): **15** direct mentions.
- Operational verbs (calculate/build/display) are very frequent: about **77** occurrences.

## Assignment Template To Imitate
1. Present the dataset and variables with units of measurement.
2. Ask for descriptive indicators (mean/standard deviation/percentiles/dispersion).
3. Include a frequency distribution with a number of classes and a comparison of sample mean vs distribution mean.
4. Add a bivariate visualization or a time series.
5. Request regression (parameters, R², RMSE) and a forecast with a comment on reliability.
6. Conclude with at least one short interpretative text request.

## Language Register
- Formal, technical register, with direct imperatives ("Calculate", "Build", "Display", "Explain").
- Dense but precise instructions, without frills, focused on operability in Excel/Calc.
- A/B/make-up variants built on the same schema, with modified numerical parameters.
